# GitHub User List App

A React application that fetches and displays a list of GitHub users using the GitHub public API. Built to demonstrate the use of React hooks such as useEffect and useState along with component-based architecture.

---

## Features
- Fetches real-time data from the GitHub public API
- Displays users in a clean, responsive card layout
- Uses useEffect for API calls
- Uses reusable React components
- Opens GitHub profiles in a new tab
- Modern UI with hover effects
- Beginner-friendly and well-structured code

---

## Tech Stack
- React (Create React App)
- JavaScript (ES6+)
- CSS
- GitHub REST API
